История создания языков программирования

Python создан в 1989-1991 годах голландским программистом Гвидо ван Россумом.

C разработан в 1969-1973 годах сотрудником Bell Labs Денисом Ритчи.

Java разработанв 1990-1996 годах канадским программистом Джеймсом Гослингом.

JavaScript разработан в 1996 году, автор - американский программист Брендан Айк.
